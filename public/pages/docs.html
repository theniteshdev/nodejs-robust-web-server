Quick start

  1. Install dependencies (none required for this simple server).
  2. Start the server: npm start or node server.js
  3. Open http://localhost:3000 in your browser.

Contact form

The contact form in the demo only shows client-side feedback. To handle submissions, add a POST endpoint (e.g., /api/contact) and forward messages to email or a queue.
