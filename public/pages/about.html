About Robust

Robust is a tiny, secure static-file server pattern for Node.js and a simple responsive frontend to showcase it.

Our mission

Make it easy to ship fast, secure static sites and prototypes with minimal configuration.

How it works

  • Static files live in /public.
  • Content types are set based on file extensions.
  • Path normalization prevents basic path traversal attacks.
